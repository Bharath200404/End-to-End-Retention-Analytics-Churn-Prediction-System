# 📊 Customer Churn Prediction System

An end-to-end machine learning project that predicts whether a customer is likely to churn, along with actionable business insights.

---

## 🚀 Project Overview

Customer churn is a critical problem for subscription-based businesses. This project analyzes customer behavior and builds a predictive model to identify customers at risk of leaving.

The system provides:
- Churn prediction (Yes/No)
- Churn probability score
- Key business insights

---

## 🧠 Key Features

- 📊 Exploratory Data Analysis (EDA)
- 🧹 Data Cleaning & Preprocessing
- ⚙️ Feature Engineering
- 🤖 Machine Learning Models:
  - Logistic Regression (Best Model)
  - Random Forest
  - XGBoost
- 📈 Model Evaluation (Accuracy, Recall, F1-score)
- 🌐 Streamlit Web Application
- 📉 Probability-based churn prediction

---

## 🏆 Best Model

**Logistic Regression with class balancing**

- Accuracy: ~74%
- Recall (Churn): **79%**
- Optimized for business impact (reducing missed churn customers)

---

## 📊 Key Insights

- Customers with **low tenure** are more likely to churn
- **Month-to-month contracts** have higher churn rates
- **Fiber optic users** show higher churn probability
- **Electronic check payments** are linked to higher churn
- **Senior citizens** have slightly higher churn risk
