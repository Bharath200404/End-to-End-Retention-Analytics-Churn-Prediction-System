# 📊 End-to-End Retention Analytics & Churn Prediction System

An end-to-end machine learning system that predicts customer churn and provides actionable business insights using both a web interface and a REST API.

---

## 🚀 Project Overview

Customer churn is a major challenge for subscription-based businesses. This project analyzes customer behavior and builds a predictive system to identify customers at risk of leaving.

The system includes:
- Churn prediction (Yes/No)
- Churn probability score
- Business insights for retention strategies
- Full-stack integration (Frontend + Backend)

---

## 🧠 Key Features

- 📊 Exploratory Data Analysis (EDA)
- 🧹 Data Cleaning & Preprocessing
- ⚙️ Feature Engineering
- 🤖 Machine Learning Models:
  - Logistic Regression (Best Model)
  - Random Forest
  - XGBoost
- 📈 Model Evaluation (Accuracy, Recall, F1-score)
- 🌐 Streamlit Web Application (Frontend)
- ⚙️ Flask REST API (Backend)
- 🔗 API Integration (Streamlit → Flask → Model)
- 📉 Probability-based churn prediction

---

## 🏆 Best Model

**Logistic Regression with class balancing**

- Accuracy: ~74%
- Recall (Churn): **79%**
- Focused on minimizing false negatives (missed churn customers)

---

## 📊 Key Insights

- Customers with **low tenure** are more likely to churn
- **Fiber optic internet users** show higher churn rates
- **Electronic check payments** are associated with higher churn
- **Senior citizens** have slightly higher churn probability
- Customers with **lower engagement (fewer services)** tend to churn more
